# HR-ANALYTICS-DASHBOARD---ATTRITION-ANALYSIS

## Project Overview
This project analyzes a publicly available HR performance dataset using PowerBI  to identify factors that influence HR performance and present insights through interactive dashboards.

## Tools Used
* PowerBI
* GitHub

## Dataset
* Publicly available HR Performance Dataset

## Project Objectives
* Analyze student Employees performance.
* Explore the relattionship attrition count by salary slab and others.
* Build interactive dashboards.
* Generate insights and recommendations.

## Dashboards
1. HR Attrition Performance Analysis
2. HR Performance Insights & Recommendations

## Key Insights
* Attrition by salary slab was achieved
* Attrition by business travels was achieved
* Higher percentage values for life sciences when it comes to attrition by education.
* Female employees had a higher attrition count than male.
* Hr performance shows age slabs with higher attrtion.
